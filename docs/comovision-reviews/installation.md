# Installation

## Plugin installieren

1. Melde dich im WordPress-Backend an
2. Navigiere zu **Plugins → Installieren → Plugin hochladen**
3. Wähle die ZIP-Datei `comovision-reviews-x.x.x.zip` aus
4. Klicke auf **Jetzt installieren**
5. Klicke anschließend auf **Plugin aktivieren**

## Nach der Installation

Nach der Aktivierung erscheint unter **Einstellungen → CV Reviews** das Plugin-Backend. Trage dort als nächstes deinen [Google Places API-Key und die Place ID ein](konfiguration.md).

!!! warning "Lizenzschlüssel erforderlich"
    Das Plugin benötigt einen gültigen Lizenzschlüssel um zu funktionieren. Ohne aktivierte Lizenz wird das Karussell im Frontend nicht ausgegeben. Den Schlüssel trägst du unter **Einstellungen → CV Reviews → Tab: Einstellungen** ein.

## Updates

Sobald eine neue Version verfügbar ist, erscheint die Meldung direkt im WordPress-Backend unter **Dashboard → Updates** – genauso wie bei jedem anderen Plugin. Ein Klick auf **Aktualisieren** genügt.

Voraussetzung: Der Lizenzschlüssel ist aktiv und die Domain ist auf dem Lizenzserver freigeschaltet.
