# Installation

## Plugin installieren

1. Melde dich im WordPress-Backend an
2. Navigiere zu **Plugins → Installieren → Plugin hochladen**
3. Klicke auf **Datei auswählen** und wähle die ZIP-Datei `comovision-reviews-x.x.x.zip`
4. Klicke auf **Jetzt installieren**
5. Klicke anschließend auf **Plugin aktivieren**

Nach der Aktivierung erscheint unter **Einstellungen → CV Reviews** das Plugin-Backend mit drei Tabs:

| Tab | Beschreibung |
|---|---|
| **Einstellungen** | API-Key, Place ID, Cache, Lizenzschlüssel |
| **Bewertungen verwalten** | Bewertungen ein-/ausblenden, sortieren, Keywords |
| **Anpassen** | Design, Layout, Farben, Typographie |

## Nächste Schritte

1. [API-Key und Place ID eintragen](konfiguration.md)
2. [Lizenzschlüssel aktivieren](lizenz.md)
3. Shortcode `[comovision_reviews]` in eine Seite einfügen

!!! warning "Lizenzschlüssel erforderlich"
    Das Plugin benötigt einen gültigen Lizenzschlüssel um zu funktionieren. Ohne aktivierte Lizenz wird das Karussell im Frontend nicht ausgegeben. Den Schlüssel trägst du unter **Einstellungen → CV Reviews → Einstellungen** ein.

## Updates

Sobald eine neue Version verfügbar ist, erscheint die Meldung direkt im WordPress-Backend unter **Dashboard → Updates** – genauso wie bei jedem anderen Plugin. Ein Klick auf **Aktualisieren** genügt.

Voraussetzung: Der Lizenzschlüssel ist aktiv und die Domain ist auf dem Lizenzserver freigeschaltet.
