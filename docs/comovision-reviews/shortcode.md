# Shortcode

## Verwendung

```
[comovision_reviews]
```

Den Shortcode einfach in den Inhalt einer Seite, eines Beitrags oder eines Widgets einfügen.

## Was wird angezeigt

- Bewertungs-Karussell mit den im Review Manager freigegebenen Bewertungen
- Profilbild, Name, Sternebewertung, Datum und Bewertungstext je Slide
- Navigation per Pfeil, Swipe (Touchscreen) oder Tastatur
- Gesamtbewertung und Anzahl der Google-Bewertungen als Fußzeile
- Google-Logo

## Verhalten

- Das Karussell startet automatisch und pausiert beim Hover
- Die Höhe passt sich automatisch an die längste Bewertung an (kein Springen)
- Profilbilder werden lokal gecacht – keine externen Anfragen an Google beim Seitenaufruf

## Voraussetzungen

!!! warning
    Ohne aktiven Lizenzschlüssel wird das Karussell nicht ausgegeben.
